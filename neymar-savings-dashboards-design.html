What v2 locks in

  • Neymar Savings navy and gold across both dashboards, matching the two mobile apps exactly.
  • Agent identity is real: the Agent ID is locked (shown with a lock chip), the name is editable by Admin and Super Admin, and a suspended or terminated agent triggers a customer-reassignment flow instead of a delete.
  • Money rules are enforced on screen: platform commission is tracked as its own figure, and withdrawals that would breach the two-contribution floor are blocked, not just discouraged, with the block itself recorded in the audit log.
Design phase complete: Agent app, Customer app, Admin and Super Admin dashboards, all in one Neymar Savings system.
